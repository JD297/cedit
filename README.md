## Requirements
#### Linux Ubuntu
    sudo apt update && sudo apt install build-essential make libncurses5-dev

#### FreeBSD
    pkg install gcc ncurses libncurses-dev

## Make commands
#### build
    make
#### clean build
    make clean

#### install build
    make install

#### uninstall build
    make uninstall
